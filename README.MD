# JS MUSIC PLAYER

Second JS Project

## Functionality:

- Keyframe animation
- use of ::after(pseudo element)
- HTML5 AUDIO API
  - audio.play()
  - audio.pause()
  - audio.duration
  - event: timeupdate, ended
  - For every event there is a Event Details

## VS Code Short cut

- select a line of code
- then, use ALT + arrow key
- to move code up down
